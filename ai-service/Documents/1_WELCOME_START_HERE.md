### Key Terms You'll Learn
Don't worry if you don't know these yet! We'll explain them all:

- **Frontend** = What users see in their web browser
- **Backend** = The server that processes requests
- **Database** = Where all the data is stored
- **API** = How different parts of the system talk to each other
- **Docker** = A container that packages code so it works everywhere
- **Microservice** = A small, specialized service (like our AI service)
- **Authentication** = Login/password system to verify who you are

## 💡 A Quick Analogy

Think of Project Rewood like a **restaurant**:

```
┌─────────────────────────────────────────────────────────┐
│ CUSTOMER (FRONTEND)                                     │
│ - Sees the menu on the website                          │
│ - Orders food                                           │
│ - Reads the bill                                        │
└─────────────────────────────────────────────────────────┘
                           ↕️
┌─────────────────────────────────────────────────────────┐
│ WAITER (API)                                            │
│ - Takes the order from customer                         │
│ - Passes order to kitchen                               │
│ - Brings back the meal                                  │
└─────────────────────────────────────────────────────────┘
                           ↕️
┌─────────────────────────────────────────────────────────┐
│ KITCHEN (BACKEND)                                       │
│ - Receives order from waiter                            │
│ - Checks pantry (database) for ingredients              │
│ - Prepares the meal                                     │
│ - Calls the waiter when ready                           │
└─────────────────────────────────────────────────────────┘
                           ↕️
┌─────────────────────────────────────────────────────────┐
│ PANTRY (DATABASE)                                       │
│ - Stores all ingredients                                │
│ - Kitchen searches for what they need                   │
└─────────────────────────────────────────────────────────┘
```

This is exactly how web applications work!
